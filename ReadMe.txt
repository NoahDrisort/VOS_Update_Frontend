**FOR UBUNTU**
- Extract file icu4c-64_2-src.tgz
- Copy all folders in extracted folder to current folder. (OVERWRITE ALL FOLDERS: bin, include, lib, sbin, share)
- Set LD_LIBRARY_PATH to "lib" folder in current folder.
- Run make main to compile
- Reading text from "input.txt" and print to "output.txt"
